# Wireshark Notes

## What is Wireshark?

Wireshark is a network protocol analyzer that captures and displays network traffic in real time.

---

## Why Learn Wireshark?

- Analyze network traffic.
- Troubleshoot network issues.
- Detect suspicious activity.
- Understand how devices communicate.

---

## Basic Terms

### Packet

A small unit of data sent across a network.

### Capture

Recording packets moving across a network.

### Protocol

Rules that devices follow to communicate.

Examples:

- HTTP
- HTTPS
- DNS
- TCP
- UDP
- ICMP

---

## Protocols I Know

### DNS

Converts domain names into IP addresses.

### HTTP

Transfers web pages.

### HTTPS

Secure version of HTTP.

### TCP

Reliable communication.

### UDP

Fast communication with no delivery guarantee.

### ICMP

Used by commands like ping.
